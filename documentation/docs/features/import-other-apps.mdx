# Import from Other Apps

NotallyX provides the possibility to import your existing notes from other apps.

## To Import from other Apps
1. **Open the Settings**
2. Under **Backup**, click **Import Notes from other App**
3. Select the app you want to import from to get further instructions

## Available Imports
- [Notally](https://github.com/OmGodse/Notally)
- [Google Keep](https://keep.google.com/)
- [Evernote](https://evernote.com/)
- [Quillpad](https://quillpad.github.io/)
- Plain Text Files
- Markdown Files
- JSON Files
