NotallyX — минималистичное приложение для заметок с простым дизайном и мощными возможностями.

Организация

• Создание заметок с форматированным текстом (жирный, курсив, моноширинный шрифт и другое)
• Создание списков задач с поддержкой подзадач
• Установка напоминаний с уведомлениями для важных заметок
• Добавление файлов к заметкам: изображений, PDF и других
• Сортировка заметок по названию, дате изменения или дате создания
• Отображение заметок в виде списка или сетки
• Цветовые метки, закрепление и ярлыки для быстрой организации
• Добавление кликабельных ссылок (телефон, email, веб-адреса)
• Быстрое удаление выполненных задач
• Отмена и повтор действий
• Широкие настройки интерфейса
• Быстрая отправка заметок как текста
• Блокировка заметок через биометрию или PIN
• Настраиваемые автоматические резервные копии
• Создание аудиозаметок
• Адаптивная иконка приложения Android
• Поддержка устройств начиная с Android Lollipop

Удобство

• Тёмная тема
• Полностью бесплатное приложение без рекламы
• Настраиваемый размер текста
• Автосохранение и автоматические резервные копии (при желании с паролем)
• Добавление заметок и списков на главный экран через виджеты
• Импорт заметок из других приложений, например Evernote, Google Keep или из обычных текстовых файлов

Приватность

В приложении нет рекламы, трекеров или аналитики. Все ваши заметки хранятся только на устройстве и никогда его не покидают.

Дополнительные разрешения

• Уведомления и фоновая служба — для отображения уведомлений о напоминаниях или ошибках резервного копирования

• Точные будильники — для установки напоминаний на точное время

• Запрет перехода телефона в сон и запуск при старте системы — используются для автоматического резервного копирования после перезапуска устройства

• Запись аудио — для создания аудиозаметок

• Биометрия — используется для разблокировки приложения при включённой защите

Примечание

Из-за ошибки в системе Xiaomi некоторые устройства с MIUI могут не показывать параметры форматирования текста.
